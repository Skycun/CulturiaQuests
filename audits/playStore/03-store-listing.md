# 3. Store Listing (Fiche Play Store)

## Elements requis

### 3.1 Textes

| Element | Contrainte | Statut |
|---------|-----------|--------|
| Titre de l'app | Max 30 caracteres | Suggestion : "CulturiaQuests" (14 car.) |
| Description courte | Max 80 caracteres | A REDIGER |
| Description longue | Max 4000 caracteres | A REDIGER |

**Suggestion de description courte :**
> RPG geolocalise culturel : explorez, combattez et collectionnez en France !

**Points a couvrir dans la description longue :**
- Concept du jeu (RPG geolocalise, exploration culturelle)
- Fonctionnalites principales (carte, expeditions, quiz, badges, musees)
- Aspect educatif/culturel
- Gratuit sans achats in-app
- Cadre academique (si pertinent pour la credibilite)

### 3.2 Screenshots

**Minimum** : 2 screenshots
**Recommande** : 8 screenshots

| Type | Dimensions | Requis |
|------|-----------|--------|
| Telephone | 1080x1920 (portrait) | OUI (min 2) |
| Tablette 7" | 1200x1920 | RECOMMANDE |
| Tablette 10" | 1600x2560 | RECOMMANDE |

**Pages a capturer en priorite :**
1. La carte avec les marqueurs POI/musees
2. L'ecran d'expedition (combat)
3. Le quiz quotidien
4. L'equipement du personnage
5. La page de badges/collection
6. L'interaction avec un PNJ
7. Le coffre de butin
8. Le profil/guild

**Conseils :**
- Utiliser un emulateur Android avec de belles donnees de demo
- Ajouter du texte overlay sur les screenshots pour expliquer les features (optionnel mais efficace)
- Utiliser des outils comme [screenshots.pro](https://screenshots.pro) ou Figma pour les mockups

### 3.3 Feature Graphic

- **Dimensions** : 1024x500 pixels
- **Format** : JPG ou PNG 24 bits (pas de transparence)
- Affichee en banniere en haut de la fiche
- Doit representer visuellement l'univers du jeu

### 3.4 Icone Play Store

- **Dimensions** : 512x512 pixels
- **Format** : PNG 32 bits
- Meme design que l'icone de l'app (voir 01-icones-et-assets.md)

### 3.5 Categorie et classification

| Parametre | Valeur suggeree |
|-----------|----------------|
| Type d'application | Jeu |
| Categorie | Jeux de role |
| Classification IARC | PEGI 7 ou Everyone 10+ (E10+) |
| Contient des pubs | Non |
| Achats in-app | Non |

### 3.6 Questionnaire IARC

A remplir dans la Google Play Console. Points importants :
- Le jeu contient des interactions sociales (amities entre joueurs)
- Le jeu utilise la geolocalisation
- Le jeu ne contient pas de violence graphique (systeme de DPS abstrait)
- Le jeu ne contient pas de contenu genere par les utilisateurs visible publiquement (sauf pseudonymes)
- Pas de jeux d'argent, pas de contenu sexuel, pas de substances

### 3.7 Coordonnees de contact

A renseigner dans la Play Console :
- Email de support (obligatoire)
- Site web (optionnel mais recommande)
- Telephone (optionnel)
